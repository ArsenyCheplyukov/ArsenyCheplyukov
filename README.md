# Hi, I'm Arseny 👋

**Python engineer — backend & data.** By day I build production APIs and data pipelines: FastAPI, PostgreSQL, Kafka, Redis, AWS, and lately Databricks/Spark on a lakehouse platform. By night I run a fully **self-hosted RAG stack** — my own LLM inference, embedding and reranking services wired into a document-search engine, no cloud APIs required.

Before all that I taught microcontrollers to do useful things: an ultrasonic flaw detector, a robotic arm with on-device inverse kinematics, a dosimeter, telemetry tooling for a university satellite. That's where my habit of making machines *actually work* comes from.

## 🔭 Building now

- **norma-search** — self-hosted RAG search engine for regulatory documents: hybrid retrieval (vector + full-text, RRF), cross-encoder reranking, bounded multi-hop, ~770 backend tests. *(private while it's raw — ask me for a demo)*
- [**ollama-serve**](https://github.com/ArsenyCheplyukov/ollama-serve) — lifecycle wrapper for local LLM inference: OpenAI-compatible API, VRAM hygiene, systemd
- [**embed-serve**](https://github.com/ArsenyCheplyukov/embed-serve) — local embedding service for RAG pipelines (ONNX / PyTorch backends)
- [**reranker-serve**](https://github.com/ArsenyCheplyukov/reranker-serve) — cross-encoder reranking microservice
- [**pix2text-container**](https://github.com/ArsenyCheplyukov/pix2text-container) — dockerized OCR & math-formula extraction service
